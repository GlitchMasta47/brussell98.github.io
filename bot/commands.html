@Bot-chan Text Doing this will let you talk to the bot. Doesn't work in PMs. @Bot-chan are you real?
]help [command] Get a list of commands or information on a specific command. ]help prune h, commands
]id [channel] Get your user ID. If the suffix is "channel" get the channel's ID. ]id channel myid
]ping Check if the bot is working. Tells you response time in ms. ]ping p
]beep boop ]beep
]join <invite(s)> [-a] Invite the bot to a server. If "-a" is used it will show the join message. ]join invite link -a j, joins
]about Display info about the bot. ]about
]letsplay [game] Ask if anyone wants to play a game. ]letsplay CS:GO lp, play
]dice [(rolls)d(sides)] Roll a dice with 6 sides or as specified. ]dice 2d12
]roll [max] Roll a random number between 1-max or 1-100. ]roll 1-10
]info [user] Get info on the server, or, if specified, a user ]info GenericUsername i
]choose <option>, <option>, [option], ... Makes a choice for you. ]choose sleep, discord c, pick
]lotto end | enter | new [max entries] [-noautoend] | <pick from> | everyone Start / stop / enter a lottery. Allows setting max entries per user, auto end, predefined entries, and picking from all members. ]lotto new 2 -noautoend ]lotto @Brussell @Not-Brussell
]strawpoll <option>, <option>, [option], ... Create a strawpoll. ]strawpoll DOTA2, LoL straw, poll
]vote new [topic] | +/- | end Vote on something. ]vote new Ban @GenericUsername v
]8ball [question] Ask the magic 8 ball. ]8ball 8, ball
