Command List

Key: [optional] <required>


About

Shows you some info about Mirai. Also contains some useful links.
]about
Coldown: 5 seconds

Choose

Have Mirai make a tough decision for you.
]choose <choice> | <choice> [| choice...]
Aliases: c, pick, decide
Coldown: 2 seconds

Invite

Get the link to add the bot to another server.
]invite
Aliases: oauth
Coldown: 5 seconds

Ping

Check if the bot is working. It also returns the delay in response.
]ping
Aliases: p
Coldown: 2 seconds

Random Comic

Grab a random comic from explosm.net/rcg.
]randomcomic
Aliases: rcg, comic
Coldown: 10 seconds

Settings

Adjust the settings for how the bot works on the server. More information here: Settings.
See page for usage
Aliases: set, config
Coldown: 3 seconds
