I usually develop this bot when I'm tired and I don't test anything so you know it's going to be great.

fast_forward

Good Response Times

Bot-chan is hosted on a server instead of my home PC. This means that you don't have to worry about the bot being offline or being slow to respond. (Does not account for other APIs slowing it down)

lock

Permission Based

All commands are based on the permissions of the user. This means the bot can be used without any setup. Of course, if you want to use all features of the bot head over to the settings page.

settings

Customizable

You can change the settings of the bot for each server. Some things include ignoring channels, custom welcome messages, and ban alerts. Head over to the settings page for more information.



Features

chat_bubble

Cleverbot

By far the most used feature, the cleverbot module allows you to have conversations with the bot. Just do @Bot-chan <your text here>.

announcement

Server Announcements

Server moderators can use the bot to send a message to all users on the server via a private message.

search

My Anime List Search

The bot can get information on an anime or manga from MAL.

color_lens

Role colors

Want to give users custom role colors? You can use the bot to set the hex color of a role or give a user their own color.

videogame_asset

Osu! Commands

The bot features four osu! commands. It can generate an osu!next styled signature for a user. It can also fetch a user's stats and best or recent plays.

star

Rate Waifu

Have the bot rate waifus! This is based on scores from the best girl contests. It features auto-complete so it will try to match partial names. (This also works on users)

star_border

Lotteries

Start a lottery that users can enter and the bot will choose a winner from the entries. You can also specifiy how many times users can enter. Great for giveaways.

remove_circle

Kick/Ban with reason

You can provide a reason or message that the user will recieve. Great for mobile until Discord catches up on their mobile apps.

chat_bubble

Mute Users

Give a user the "muted" role (even from mobile) for a specified ammount of time. After that time is up they will become unmuted.
