Bot-chan Server Settings


Welcome Message

This is displayed when a user joins the server. It has the following parameters:
$SERVER$ The name of the server
$USER$ The name of the user that joined

}settings welcomemsg **NEW FRIEND DETECTED:** $User$
}settings disable welcomemsg
}settings enable welcomemsg

Ban Alerts

When this is enabled the bot will send a message in #general notifying that a user was banned. It will also send a direct message to the banned user letting them know that they were banned.

}settings enable banalerts
}settings disable banalerts

Delete Commands

If this is enabled and the bot can manage messages, the bot will delete commands leaving only the responses.
Note: Commands like ]choose and ]rps will not be removed as the command is relevant.

}settings enable deletecmds
}settings disable deletecmds

Name Changes

When enabled, the bot will send a message in #general when a user changes their username. The message will show their old username and new username.

}settings enable namechanges
}settings disable namechanges
